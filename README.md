# TypeForge

TypeForge is a premium typing performance analyzer designed as a polished, modern single-page web experience. It helps users measure typing speed, accuracy, mistakes, and a custom Forge Score in a refined interface inspired by modern developer tools.

## Features

- Premium landing experience with animated gradients and glassy cards
- 60-second typing test with live metrics
- Two typing modes: Classic and Developer
- Randomized paragraph and code snippet prompts
- Smooth results screen with animated score reveal
- Fully responsive layout for desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Installation

1. Open the project folder in VS Code.
2. Start Live Server on the project root.
3. Open the generated local URL in your browser.
